## Featured Projects

### Abhyas
**AI-Powered Examination Platform**

A full-stack examination platform that leverages AI to automate question generation, evaluate descriptive and coding answers, and manage assessments for educational institutions.

**Highlights**
- AI evaluation for subjective and coding answers
- AI-generated MCQs and question papers
- Image-based question understanding
- Role-based authentication (Super Admin, Admin, Teacher, Student)
- Batch AI grading pipeline for fast evaluations
- Analytics and performance tracking

**Tech Stack**

Next.js • React • TypeScript • Node.js • Express • PostgreSQL • Drizzle ORM • OpenAI API

**AI Workflow Automation Platform**

Built during a hackathon using the Corsair SDK, GinnieAI acts as an AI assistant capable of managing emails, scheduling meetings, and executing workflows from a unified chat interface.

**Highlights**
- AI email assistant
- Meeting scheduling
- AI tool calling
- Workflow automation
- Secure API integrations
- Razorpay payment integration

**Tech Stack**

Next.js • Node.js • PostgreSQL • OpenAI • Corsair SDK
